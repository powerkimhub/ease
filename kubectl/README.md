
# Usage

## node status
    $ lsnode
    $ descnode
    $ descnode ip-172-31-20-12

## pod status

    $ lspod
    $ descpod
    $ descpod mysql

## deployment status
    $ lsdeploy
    $ descdeploy
    $ descdeploy mysql

## service status
    $ lssvc
    $ descsvc
    $ descsvc mysql

## image status
    $ lsimg
    $ lsimg mysql
